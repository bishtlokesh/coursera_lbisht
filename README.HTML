README: HOW THE MODEL WAS BUILT

FOLLOWING STEPS:

1) THE DATA WAS LOADED INTO R DATASET

2) VISUAL ANALYSIS OF RAW DATA SHOWED LOT OF COLUMNS WITH NA OR BLANKS WHICH WERE NOT CONTRIBUTING TO THE RESULT. THESE COLUMNS WERE ELIMINATED FROM THE DATA.

3) ON FURTHER EXPLORATORY ANALYSIS, SOME ADDITIONAL COLUMNS LIKE user_name WERE ELIMIATED.

4) AFTER STEP 3, WE HAD 54 COLUMNS INCLUDING THE OUTPUT COLUMN.

5) THE DATA WAS THEN SPLIT INTO 3 SETS:

TRAIN (60%)
TEST(20%)
VALIDATION(20%)

6) THE TRAIN DATASET WAS THEN PREPROCESSED IN FOLLOWING WAYS:

REMOVING NEAR ZERO VARIANCE VARIABLES
INSPECTING CORRELATION
APPLYING PCA WITH 90% ACCURARY, RESULTED IN 19 PCAS NEEDED

7) RANDOM FOREST ALGORITHM WAS USED FOR BUILDING THE MODEL.

8) THIS GAVE 100% ACCURACY IN TRAIN, TEST AND VALIDATION WITH 1 MISS IN THE FINAL TEST.
